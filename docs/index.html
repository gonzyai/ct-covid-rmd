Summary

This site contains COVID 19 Data for the 8 counties in Connecticut along with graphs depicting the daily number of cases over time, the daily number of deaths over time, and the daily number of hospitalizations over time.

See the repo for the source code. HTML is built using RMarkdown and flexdashboard.
